modal.js

This is a open-source project. Fork it.

Tiny, powerful and highly extensible plugin to handle modals.

This is a really tiny plugin. Less than 100 lines of code. Only two files. 2 kilobytes (500 bytes minified and gzipped). Tiny, right?

How to install?

Good question. First, make sure that you have the code. You can download a .zip file, .tar.gz file, or you can download in the hacker way:

git clone https://github.com/hangnest/modal.js.git

Now, you have to include the code into your project. Adding this:

<link href="stylesheets/modal.css" rel="stylesheet" type="text/css" />
<script src="javascripts/modal.jquery.js"></script>

You are ready to start using it.

How to use?

This is why you are here. The ideal scenario is to put a unique class (or id) to all the elements that are able to open a modal. But remember, this is a flexible plugin. To make the ideal scenario, you can use this code:

<script type="text/javascript">
	$(".openModal").modal();
</script>

You got it? Nice! But... what happens if this is not the ideal scenario? Don't worry, there is a solution for that.

Suppose that you have to open a modal when something happens, like a AJAX callback or something like that. You can open the modal, in a really easy way:

$(".openModal").modal({
	action: "open",
	showElement: $(".openModal")
});

When you use a selector to call itself, that action is called self invoke. Not only in this plugin, is a popular term.

As the same way that you can open the modal, you can close. Yes, remotely. Also you can set the class name (or id) to the close button.

What about the settings?

The settings or parameters that this plugin uses are really simple. Take a look:

First of all you can call the function without set any parameters. And that is cool. But if you want, the functions uses only 4 parameters.

showElement: this means the name (class or id) of the selector that you are trying to show.

closeButton: this means the name (class or id) of the selector that are able to close the modal (showElement).

action: custom action to open or close the modal (showElement) in a "manual" way.

subFunction: this is really important. Allows you to execute a function inside the plugin. Used for many purposes, for example load content dynamically via AJAX.

To set the parameters that you want, is easy, like the previous example:

$(".buttonToOpen").modal({
	showElement: $(".myCustomElement"),
	closeButton: $(".myCustomButton"),
	action: "open",
	showElement: $(".openModal")
});

Remember when I say "highly extensible", well, add your own parameters and configuration is really simple. Take a look to modal.js file and start creating your own awesome modals.

#1 Getting Started

We are to use the follow code:

<a href="#" onclick="return false;" modalToShow="#customModal" class="clickToOpen">Show my a cute puppy</a>
<div id="customModal" class="modal">
	<img class="cont" src="images/puppy.jpeg" border="0" />
	<a class="close">Close</a>
</div>
<script type="text/javascript">$(".clickToOpen").modal();</script>

Remember that I am using my own style, the plugin only include the basic style to work.

#2 Using an action

<a href="#" onclick="inOneSecond(); return false;" class="clickToRun">Show my a cute puppy in 1 second</a>
<div id="customModal" class="modal">
	<img class="cont" src="images/puppy.jpeg" border="0" />
	<a class="close">Close</a>
</div>
<script type="text/javascript">
	function inOneSecond() {
		setTimeout(function() {
			$("#customModal").modal({
				action: "open",
				showElement: $("#customModal")
			});
		}, 1000);
	}
</script>
Show my a cute puppy in 1 second

#3 Using an function

<a href="#" onclick="return false;" modalToShow="#newModal" class="loadPuppy">Show my a different puppy and when it's loaded alert me the width</a>
<div id="newModal" class="modal">
	<img class="cont" id="otherDog" src="images/puppy2.jpg" border="0" />
	<a class="close">Close</a>
</div>
<script type="text/javascript">
	var puppyFunction = function() {
		setTimeout(function() {
			alert($("#otherDog").load().width() +"px. Really cute, right?");
		}, 200);
	};
	$(".loadPuppy").modal({
		subFunction: puppyFunction
	});
</script>
